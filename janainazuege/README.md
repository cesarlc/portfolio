## 🚀 Como Usar

O projeto está configurado para rodar com Vite. Use o servidor de desenvolvimento para acompanhar as alterações localmente.

Para adicionar a foto de perfil real:
1. Salve a foto como `profile.jpg` na pasta `src/imports/`
2. Atualize o componente `ProfileAvatar` para usar o caminho correto
